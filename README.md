# 🎓 Student Sign Up Portal

A simple and responsive **Student Sign Up Portal** developed using **HTML** and **CSS**. This project is designed for beginners to understand the fundamentals of web development and basic user interface design.

## 📌 Features

- Student Sign Up page
- Clean and responsive UI
- Modern color palette
- Easy navigation between pages
- Beginner-friendly project structure
- Simple and readable code

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
Student-Sign-Up-Portal/
│── index.html
│── home.html
│── sign up.html
│── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Student-Sign-Up-Portal.git
```

2. Open the project folder.

3. Run `index.html` in your preferred web browser.

## 📖 Learning Objectives

This project helps beginners learn:

- HTML page structure
- CSS styling
- Navigation using hyperlinks
- Tables in HTML
- Responsive layout basics
- Organizing a simple web project
